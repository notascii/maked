### To perf measure for pingpong

starting locally

`bash ssh <login@access.grid5000.fr>`

`bash grenoble`

`bash cd commands`

`bash ./create_nodes.sh`

Open a new terminal and access grenoble's site again

`bash cd commands`

`bash ./installgo.sh`

`bash ./measure_perf.sh`

Check result in `../local_perf`
